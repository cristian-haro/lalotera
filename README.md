# lalotera

Este proyecto, llamado LaLotera, es una aplicación sencilla diseñada para gestionar partidas de lotería o bingo desde tu ordenador.

- Saca números al azar: La aplicación funciona como un "bombo virtual" que extrae números del 1 al 100 de forma aleatoria, asegurando que no se repitan durante la partida.
- Visualización clara: Muestra el número actual en un tamaño muy grande en el centro de la pantalla, ideal para que todos los jugadores puedan verlo fácilmente.
- Tablero de control: Incluye un panel con todos los números posibles. Cuando sale un número, este se marca automáticamente en color amarillo en el tablero para que sepas en todo momento qué números han salido ya.
- Interfaz moderna: Tiene un diseño oscuro y limpio ("Modo Oscuro") que hace que sea cómodo a la vista y profesional.
- Gestión de partidas: Incluye un botón para reiniciar el juego, lo que limpia el tablero y vuelve a mezclar todos los números para empezar una nueva ronda.
- En resumen: es la herramienta perfecta para organizar una partida de lotería con amigos o familia sin necesidad de tener las bolas y el bombo físico.
